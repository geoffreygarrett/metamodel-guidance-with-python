mmgpy.sampling¶

Sampling methods

sampling.full_factorial(bounds, n_sample_tuple)

Custom Grid (CG): This Grid method is the sample as the UG method, but allows for non-uniform grid sampling.
sampling.uniform_grid(bounds, n_sample, flatten=True, edges=False, form='sx', **kwargs)¶

Uniform Grid (UG): Uniform grid (UG) is the simplest sampling method in which the samples are created using node points at fixed intervals uniformly distributed for every dimension.

Notes
